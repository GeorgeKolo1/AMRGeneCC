# AMRGeneCC
This repository contains a simple CLI software tool to concatenate the output of all AMR genes and antibiotics generated by NCBI AmrFinder across all assemblies within a folder.

## Installation
You can install this package by opening the terminal/windows powershell etc and following the instructions below

### Git clone & pip install

It is recommended to do this in a conda environment or virtual environment to prevent conflicts between dependencies.
**Make sure you have pip installed.** To install pip in a conda environment you can use the following command: ```conda install pip```

1.

```git clone https://github.com/GeorgeKolo1/AMRGeneCC.git```

2.

```pip install .```

### Testing the installation

You can test the installation by running <amrgene --help or amrgene -h>

## Example Usage

```amrgene PATH/TO/YOUR/FILES/```

## Input

The amrgeneCC tool is very simple, it takes only a single input which is the path to your folder containing the results files from NCBI AmrFinder

## Output

The amrgeneCC tool outputs two CSV files:

    AMR_genes_results.csv: a file that contains each isolate as a column with the rows containing individual AMR genes identified
    AMR_antibiotics_results.csv: a file that contains each isolate as a column with the rows containing individual antibiotics that the isolate is predicted to be resistant to


The output files will be saved to the directory/folder that you provide the amrgene function with
